# Block Puzzle Game

A simple browser-based block puzzle game built with HTML, CSS, and JavaScript.

## Files

- `index.html` - game UI and layout
- `styles.css` - game styling
- `script.js` - game logic

## Run locally

1. Open `index.html` in a web browser.
2. Or use a local server such as Live Server in VS Code.

## Notes

- No build step is required.
- This repo contains a lightweight static web game.
